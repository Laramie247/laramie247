Domain Migration Checklist

From Bubble (optional) to Vercel via Name.com

0% Complete

Cheat Sheet: What are these terms?
A Record

Points your root domain (example.com) to a specific server IP address.

CNAME Record

Points a subdomain (www.example.com) to another domain name. Acts like a forwarder.

Nameservers

The "master switch" for your domain. Must be set to Name.com Default for these steps to work.

1. Bubble.io (Cleanup)

  • Go to Settings > Domain/Email > "Delete this domain name".

2. Vercel (Get Config)

  • Settings > Domains > Add `yourdomain.com`.
  • Vercel will ask to add the www version automatically. Say yes.
  • Note down the A Record IP (e.g., 76.76.21.21) and CNAME Value.

3. Name.com (Update DNS)

  • Must be "Name.com Nameservers". If Custom, switch back to Default.
  • Delete existing A records (@) or CNAME (www) pointing to Bubble.
  • Type: A | Host: blank or @ | Answer: [Vercel IP]
  • Type: CNAME | Host: www | Answer: cname.vercel-dns.com

4. Verification

  • Wait for the error to turn into a success indicator.
